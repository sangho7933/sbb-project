1. 던전 (원정)

  • 전투력 상승의 핵심은 장비 파밍
  • 정복 크라오 동굴 → 목걸이, 투구
  • 정복 드라웁니르 → 하의, 신발
  • 부족하면 탐험 드라웁니르로 유일 장비 확보

2. 내실 (탐험 + 지역 퀘스트)

  • 물음표 → 봉인 던전 / 주둔지
  • 초록 별 → 지역 퀘스트
  • 데바니온 포인트 1당 전투력 +1
  • 흔적 수집은 후순위

3. 강화 & 마석

  • +10강까지는 100% 성공
  • 강화 시 전투력 +1~+2 증가
  • 마석/영석 각인 시 +1~+4 증가
  • 전부위 각인 시 큰 전투력 상승 가능

4. 슈고 페스타

  • 매시간 15분, 45분 진행
  • 유일 장비 획득 가능
  • 데바니온 결정 구매 가능

5. 추가 콘텐츠

  • 악몽 → 아리엘 결정 획득
  • 각성전 / 토벌전 → 재료 획득
  • 어비스 상점 → 장비 or 스킬 선택
  • 메인 퀘스트 완료 후 유일 장비 강화만으로 약 1200 도달
  • 이후 목표는 1600 이상 전투력
  • 장비 + 내실 + 강화 + 콘텐츠 병행이 핵심
  • 장비 파밍 → 내실 → 강화 순으로 진행
  • 던전 + 내실 병행이 가장 빠른 성장 루트
  • 1600 전투력까지는 꾸준히 올릴 수 있음
